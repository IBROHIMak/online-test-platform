# Assets Papkasi

Bu papkada loyiha uchun kerakli rasmlar va ikonkalar saqlanadi.

## Struktura

```
assets/
├── images/          # Rasmlar (screenshots, backgrounds, etc.)
│   ├── logo.png
│   ├── banner.jpg
│   └── ...
│
└── icons/           # Ikonkalar (favicon, app icons, etc.)
    ├── favicon.ico
    ├── icon-192.png
    └── icon-512.png
```

## Tavsiya etiladigan formatlar

- **Logo**: PNG (transparent background)
- **Icons**: SVG yoki PNG
- **Screenshots**: JPG yoki PNG
- **Favicon**: ICO yoki PNG (32x32, 64x64)

## Optimizatsiya

Rasmlarni optimizatsiya qilish uchun:
- [TinyPNG](https://tinypng.com/) - PNG/JPG
- [SVGOMG](https://jakearchibald.github.io/svgomg/) - SVG
